# Pensamento Computacional
## 📌 Descrição do Repositório
Este repositório foi criado para organizar e armazenar as atividades desenvolvidas na disciplina de Pensamento Computacional.

As atividades estão organizadas por temas e datas, facilitando a navegação e o acompanhamento da evolução do aprendizado ao longo da disciplina.

## 🎯 Objetivo
Desenvolver o raciocínio lógico e a capacidade de resolução de problemas por meio dos princípios do pensamento computacional.
